# Changelog

## v1.0.0 (2026-03)

### Initial Release

- Core `LinkView_Visualizer`: Publication-quality pairwise microsynteny plots with BLASTN homology ribbons and TE annotation tracks
- Single-species and cross-species comparison modes
- Gene mode (`--gene1/--gene2`) and region mode (`--region1/--region2`)
- Automatic reverse complement detection
- SVG and PDF vector output via CairoSVG
- Bezier curve rendering for homology ribbons

### Planned (future releases)

- `MicroSynteny_Plotter`: Batch processing of multiple gene pairs
- `GD_Classifier`: Automated classification of gene duplication types
- `TE_Annotator`: TE family annotation overlay
