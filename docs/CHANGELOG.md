# Changelog

## v1.0.0 (2026-03)

### Initial Release

- `LinkView_Visualizer`: Publication-quality pairwise linkview plots with BLASTN homology ribbons and TE annotation tracks
- `MicroSynteny_Plotter`: Batch processing of multiple gene pairs
- `GD_Classifier`: Automated classification of Transposed / Segmental / Tandem duplications based on ribbon pattern
- `TE_Annotator`: TE family annotation overlay for mechanistic inference (Pack-MULE, Pack-Helitron, LTR retrotransposition)
